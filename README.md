## vue3-TODO-demo

#### 这是一个基于vue3.x版本，仿Mac系统的提醒事项的练习Demo
#### 功能说明
1. 可新增我的列表
2. 双击列表可进行修改
3. 点击列表会展示对应的计划详情
4. 可添加新的计划
5. 点击多选框会将该计划设置为完成状态
6. 双击计划可修改该计划
7. 可进行搜索
#### 效果演示

### 安装依赖
```
npm install
```
### 启动项目
```
npm run serve
```
### 项目打包
```
npm run build
```
